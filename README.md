⚡ Project Overview

This project is an IoT-based Smart Weather Monitoring System using ESP32 and DHT11 sensor, where environmental data is sent to the Adafruit IO Cloud Platform in real time.

It enables users to monitor temperature and humidity from anywhere in the world using a live dashboard.

🎯 Features

✨ Real-time Temperature Monitoring
💧 Real-time Humidity Tracking
📡 Wireless Data Transfer via ESP32 Wi-Fi
☁️ Cloud Storage using Adafruit IO
📊 Live Dashboard Visualization
🔄 Auto Refresh Sensor Updates
📱 Mobile + Web Access

🧰 Tech Stack
⚡ ESP32 Microcontroller
🌡️ DHT11 Sensor
☁️ Adafruit IO Cloud
💻 Arduino IDE
🌐 IoT Communication (MQTT/HTTP)
🔥 System Working
DHT11 Sensor → ESP32 → WiFi → Adafruit IO Cloud → Live Dashboard
Sensor collects temperature & humidity
ESP32 processes the data
Data is sent to Adafruit IO via Wi-Fi
Dashboard updates in real-time 📊
